# Changelog

All notable changes to this project are documented in **[GitHub Releases](https://github.com/bakissation/houwiya/releases)**.

Versioning and release notes are automated by [semantic-release](https://semantic-release.gitbook.io/)
from [Conventional Commits](https://www.conventionalcommits.org/) — `fix:` → patch, `feat:` → minor,
`feat!:` / `BREAKING CHANGE:` → major. This project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

There is no hand-maintained changelog: see the Releases page above for every version.
